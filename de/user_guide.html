Benutzerhandbuch: TapAlert

Eine persönliche Sicherheits-App, die mit einem einzigen Tastendruck SMS- und E-Mail-Benachrichtigungen mit Ihrem GPS-Standort an Ihre Notfallkontakte sendet.

1. Erster Start: Berechtigungen

Beim ersten Start führt Sie TapAlert durch einen Bildschirm zum Einrichten von Berechtigungen. Gewähren Sie Folgendes für die volle Funktionalität:

Erlaubnis Warum es benötigt wird
StandortBezieht Ihre GPS-Koordinaten und Adresse in Warnungen ein
Mikrofon / SpracheErmöglicht Voice-to-Text für benutzerdefinierte Warnmeldungen
KontakteErmöglicht die Auswahl von Empfängern aus dem Adressbuch Ihres Telefons
BenachrichtigungenZeigt eine Bestätigung an, wenn eine Warnung gesendet wird
SMS (nur Android)Sendet Benachrichtigungen direkt von Ihrer SIM-Karte
Telefonstatus (nur Android)Erkennt die Mobilfunkverfügbarkeit für einen intelligenten Fallback

Sie können während der Einrichtung Berechtigungen erteilen oder überspringen und diese später in den Systemeinstellungen Ihres Geräts anpassen.

2. Übersicht über den Startbildschirm

Der Startbildschirm verfügt über drei Seiten, durch die Sie durch Wischen nach links oder rechts navigieren können:

Seite Name Beschreibung
← Links Check-in-Timer Totmannschalter-Timer (nur Premium)
Center Panikmodus Große SOS- und „Ich bin in Ordnung“-Tasten – die Standardeinstellung
Rechts → Standardmodus Vorkonfigurierte Slots für Warnmeldungen

Die App wird immer im Panikmodus geöffnet (mittlere Seite).

3. Panikmodus

Die zentrale Seite ist auf maximale Einfachheit ausgelegt – ideal für Notfälle und ältere Benutzer.

  • SOS-Taste – sendet eine „SOS, ich brauche Hilfe“-Benachrichtigung mit Ihrem Standort.
  • Schaltfläche „Mir geht es gut“ – sendet den beruhigenden Status „Mir geht es gut“ an alle Kontakte.
  • Sprachtasten: Tippen Sie auf das Mikrofonsymbol neben „SOS“ oder „Ich bin in Ordnung“, um eine Sprachnachricht aufzuzeichnen. Ihre Rede wird in Text transkribiert und in die Benachrichtigung aufgenommen.
  • Notruftaste – wählt je nach Ländereinstellung sofort Ihre lokale Notrufnummer (z. B. 112, 911).

4. Standardmodus

Wischen Sie vom Panikmodus nach rechts, um auf den Standardmodus zuzugreifen. Auf dieser Seite wird eine Liste der vorkonfigurierten Alarmslots angezeigt.

  • Textfelder – bis zu 7 anpassbare Nachrichtenfelder (Standard: 3). Tippen Sie auf einen beliebigen Slot, um die Nachricht zu bearbeiten. Tippen Sie auf das Emoji-Symbol, um das Schaltflächensymbol zu ändern.
  • Voice-Slot – immer als letzter Slot verfügbar. Tippen Sie darauf, um eine Sprachnachricht in Echtzeit aufzuzeichnen. Der transkribierte Text wird als Benachrichtigung gesendet.

Die Anzahl der sichtbaren Textfelder kann unter Einstellungen > Anzahl konfigurierter Benachrichtigungen angepasst werden.

5. Empfänger verwalten

Tippen Sie auf das Personensymbol in der oberen Symbolleiste, um den Bildschirm „Empfänger“ zu öffnen.

  • Manuell hinzufügen: Tippen Sie auf „Hinzufügen“ und geben Sie eine Telefonnummer im internationalen Format ein (z. B. +31600000000).
  • Aus Kontakten hinzufügen: Tippen Sie auf das Kontaktsymbol, um direkt aus dem Adressbuch Ihres Telefons auszuwählen.
  • E-Mail hinzufügen – Fügen Sie optional jedem Empfänger eine E-Mail-Adresse hinzu, um E-Mail-Redundanz zu gewährleisten.
  • Entfernen: Wischen oder tippen Sie auf das Löschsymbol neben einem beliebigen Kontakt.

Kostenloses Kontingent: bis zu 1 Empfänger. Premium: bis zu 50 Empfänger.

iOS-Tipp: Wenn Sie nur wenige Kontakte sehen, gehen Sie zu Einstellungen > Datenschutz > Kontakte > TapAlert und wählen Sie Alle Kontakte aus.

6. Senden einer Benachrichtigung

  1. Öffnen Sie die App (oder tippen Sie auf Ihr Startbildschirm-Widget).
  2. Tippen Sie auf die Alarmschaltfläche (SOS, Mir geht es gut oder ein Standardmodus-Slot).
  3. Wenn ein Countdown konfiguriert ist, erscheint ein Dialog, der Ihnen Zeit zum Abbrechen gibt.
  4. Die App ruft Ihren aktuellen GPS-Standort ab und erstellt die Nachricht.
  5. Während des Sendens ertönt der Warnton.
  6. Es wird ein Statusbericht mit Erfolgs- oder Fehlerdetails pro Empfänger angezeigt.

Die Warnmeldung enthält Folgendes:

  • Ihr Name (falls in den Einstellungen festgelegt)
  • Der Nachrichtentext
  • Ein Google Maps-Link zu Ihrem aktuellen Standort
  • Ihr Akkustand

7. Countdown-Dialog

Vor dem Senden zeigt TapAlert einen Countdown an, um versehentliche Benachrichtigungen zu verhindern.

  • Standarddauer: 10 Sekunden. Einstellbar von 0 (sofort) bis 30 Sekunden unter Einstellungen > Countdown-Dauer.
  • Abbrechen: Tippen Sie während des Countdowns jederzeit auf ABBRECHEN, um den Vorgang abzubrechen.
  • Sofort senden: Tippen Sie auf JETZT SENDEN, um die verbleibende Verzögerung zu überspringen.

8. Check-in-Timer (Premium)

Der Check-in-Timer ist ein Totmannschalter: Wenn Sie ihn nicht rechtzeitig stornieren, sendet TapAlert automatisch eine Benachrichtigung. Nützlich beim Wandern, beim Alleinreisen oder in Situationen, in denen Sie möchten, dass jemand benachrichtigt wird, wenn Sie nicht mehr reagieren.

Zu Beginn:

  1. Wischen Sie nach links zur Seite „Check-in-Timer“.
  2. Verwenden Sie die Radauswahl, um eine Dauer (Stunden und Minuten) festzulegen.
  3. Geben Sie optional eine benutzerdefinierte Nachricht ein.
  4. Tippen Sie auf Timer starten.

Beim Laufen:

  • Der Timer zählt auf dem Bildschirm herunter.
  • Wenn weniger als 5 Minuten verbleiben, gibt TapAlert einen Warnton ab und zeigt einen Dialog an:
    • Abbrechen – verwerfen Sie die Warnung und lassen Sie den Timer weiterlaufen.
    • Stopp – Brechen Sie den Timer ab (es wird keine Warnung gesendet).
    • Neu starten – Setzen Sie den Timer auf die ursprüngliche Dauer zurück.
  • Wenn der Timer Null erreicht, ohne gestoppt oder zurückgesetzt zu werden, wird die Warnung automatisch gesendet.

Android: Der Timer läuft als Vordergrunddienst und bleibt aktiv, wenn die App im Hintergrund läuft oder der Bildschirm ausgeschaltet ist.

iOS: Der Timer wird möglicherweise nicht ausgelöst, wenn die App zwangsweise geschlossen wird. Lassen Sie die App im Hintergrund laufen.

9. Alarmverlauf

Tippen Sie auf das Uhrensymbol in der oberen Symbolleiste, um Ihren Alarmverlauf anzuzeigen.

  • Zeigt Zeitstempel, Nachricht und Übermittlungsmethode für jede Warnung an.
  • Tippen Sie auf einen beliebigen Eintrag, um Erfolgs-/Fehlerdetails und Fehlerinformationen pro Empfänger anzuzeigen.
  • Kostenloses Kontingent: Speichert die letzten 20 Warnungen. Premium: Speichert bis zu 200.
  • Löschen Sie einzelne Einträge oder löschen Sie den gesamten Verlauf über das Papierkorbsymbol.

10. Einstellungen

Tippen Sie auf das Zahnradsymbol, um die Einstellungen zu öffnen.

SMS-Methode

Android:

  • Direkte SMS – wird über Ihre SIM-Karte versendet, keine Internetverbindung erforderlich. Bevorzugt, wenn Mobilfunk verfügbar ist.
  • Twilio – sendet über die Twilio-API über das Internet (WLAN oder Daten). Wird als automatischer Fallback verwendet, wenn Direct SMS fehlschlägt.

iOS:

  • iOS-Nachrichten (manuell) – öffnet die vorab ausgefüllte native Nachrichten-App. Sie tippen manuell auf Senden. Kein Twilio-Konto erforderlich.
  • Twilio (Automatisch) – sendet automatisch über die Twilio-API. Erfordert ein Twilio-Konto.

Anweisungen zur Twilio-Einrichtung finden Sie im Twilio-Handbuch im Hilfemenü.

E-Mail-Redundanz

Konfigurieren Sie SMTP-E-Mail, um zusätzlich zu SMS auch Benachrichtigungen per E-Mail zu versenden. Nützlich, wenn SMS nicht verfügbar ist, aber eine WLAN-Verbindung besteht.

  1. Gehen Sie zu Einstellungen > E-Mail-Redundanz.
  2. Aktivieren Sie E-Mail-Redundanz.
  3. Wählen Sie einen voreingestellten Anbieter (Gmail, Outlook, Yahoo, iCloud) oder geben Sie benutzerdefinierte SMTP-Details ein.
  4. Geben Sie Ihre E-Mail-Adresse und ein App-Passwort ein (empfohlen anstelle Ihres Hauptpassworts).
  5. Tippen Sie zur Überprüfung auf E-Mail-Verbindung testen, bevor Sie sich darauf verlassen.

E-Mails werden an alle Empfänger in Ihrer Liste gesendet, für die eine E-Mail-Adresse gespeichert ist.

Standort

  • Standort einschließen – fügt jeder Warnung GPS-Koordinaten (Link zu Google Maps) hinzu.
  • Adresse einbeziehen – versucht auch, eine Straßenadresse aus Ihren Koordinaten aufzulösen.

Trigger-Methode

  • Zum Senden tippen – ein einziges Tippen startet den Countdown.
  • Drücken und Halten (5 Sekunden) – Sie müssen die Taste 5 Sekunden lang gedrückt halten. Empfohlen, um versehentliches Senden zu verhindern.

Alarm-Countdown

Legen Sie die Verzögerung fest, bevor die Warnung gesendet wird (0–30 Sekunden). Ein längerer Countdown gibt Ihnen mehr Zeit zum Abbrechen, wenn er versehentlich ausgelöst wird.

Toneinstellungen

  • Wählen Sie zwischen Plain Sounds oder Rich Sounds (oder None).
  • Klangmuster: Wählen Sie zwischen schnellem Alarm, kurzen Signaltönen, musikalischem Glockenspiel, heulender Sirene, pulsierendem Klingeln, Pfeifenorgel, metallischem Ring oder elektronischem Bienenstock.
  • Tonhöhe – Niedrig, Standard oder Hoch.
  • Lautstärke – mit dem Schieberegler anpassen.
  • SOS- und „Ich bin in Ordnung“-Benachrichtigungen verfügen über unabhängige Tonmuster.

Andere Einstellungen

  • Ihr Name – wird in ausgehenden Nachrichten angezeigt, damit die Empfänger wissen, wer die Benachrichtigung gesendet hat.
  • Countdown-Dauer – Verzögerung vor dem Senden (siehe oben).
  • Anzahl der konfigurierten Warnungen – wie viele Textfelder im Standardmodus angezeigt werden (3–7).
  • Beim Start automatisch starten – löst den Countdown sofort aus, wenn die App geöffnet wird. Aktivieren Sie dies nur, wenn Sie TapAlert über eine physische Tastenkombination starten.
  • Blinken bei Alarm: Der Bildschirm und die Taschenlampe blinken, wenn ein Alarm gesendet wird.
  • Heimatland – stellt sicher, dass lokale Telefonnummern korrekt formatiert sind.
  • App-Sprache – überschreibt die Sprache der Benutzeroberfläche (Standard: Gerätesprache).
  • App-Design – Hell, Dunkel oder Systemstandard.

11. Startbildschirm-Widget (Android)

Fügen Sie Ihrem Android-Startbildschirm ein TapAlert-Widget hinzu, um mit einem Fingertipp darauf zuzugreifen, ohne die App öffnen zu müssen.

Hinzufügen:

  1. Gehen Sie zu Einstellungen > Widget > Widget hinzufügen.
  2. Wählen Sie „Standard“ (1x1) für ein kompaktes Symbol oder „Groß“ (2x2) für eine bessere Sichtbarkeit.
  3. Befolgen Sie die Anweisungen auf dem Bildschirm, um es auf Ihrem Startbildschirm zu platzieren.

So funktioniert es:

  • Durch Tippen auf das Widget wird die App geöffnet und sofort der Alarm-Countdown gestartet.
  • Der Ton ertönt und der Countdown-Dialog erscheint – Sie können den Vorgang trotzdem abbrechen, wenn er versehentlich ausgelöst wird.

12. Benachrichtigungen auf den Telefonen der Empfänger hervorheben

Ihre Empfänger können ihre Telefone so konfigurieren, dass sie TapAlert-Nachrichten einen speziellen Klingelton geben oder „Bitte nicht stören“ umgehen.

iPhone-Empfänger:

  1. Öffnen Sie Kontakte und suchen Sie Ihren Eintrag.
  2. Tippen Sie auf Bearbeiten > Textton.
  3. Aktivieren Sie die Notfallumgehung – diese umgeht den Lautlos-Modus und die Funktion „Bitte nicht stören“.
  4. Wählen Sie einen lauten, markanten Ton.

Android-Empfänger:

  1. Öffnen Sie die Nachrichten-App und suchen Sie die Konversation.
  2. Tippen Sie auf das Menü > Details > Benachrichtigungen.
  3. Stellen Sie die Priorität auf „Hoch“ oder „Dringend“ ein.
  4. Aktivieren Sie „Nicht stören“ außer Kraft setzen.

Verwenden Sie Twilio? Ihre Kontakte erhalten Nachrichten von Ihrer Twilio-Nummer oder Absender-ID (z. B. TapAlert). Sie sollten diese Nummer/ID zu ihren Kontakten hinzufügen, damit die oben genannten Einstellungen wirksam werden.

13. Remote-Setup (Magic Link)

Ein Familienmitglied oder eine Pflegekraft kann TapAlert auf Ihrem Gerät über einen Magic Link aus der Ferne konfigurieren.

Die Pflegekraft öffnet TapAlert, wählt „Für jemand anderen einrichten“ aus, gibt die Kontakte, Nachrichtenvorlagen und optionalen Twilio-/E-Mail-Anmeldeinformationen ein und generiert dann einen Setup-Link. Wenn Sie auf Ihrem Gerät auf diesen Link tippen, wendet TapAlert automatisch die vollständige Konfiguration an.

Dies bedeutet, dass ein technisch versiertes Familienmitglied alles einrichten kann, ohne anwesend sein zu müssen – Sie tippen einfach auf einen Link.
