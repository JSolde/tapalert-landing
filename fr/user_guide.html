Guide de l'utilisateur: TapAlert

Une application de sécurité personnelle qui envoie des alertes SMS et e-mail à vos contacts d'urgence avec votre position GPS, en un seul clic.

1. Premier lancement : autorisations

Au premier lancement, TapAlert vous guidera à travers un écran de configuration des autorisations. Accordez les éléments suivants pour bénéficier de fonctionnalités complètes :

Autorisation Pourquoi c'est nécessaire
EmplacementInclut vos coordonnées GPS et votre adresse dans les alertes
Microphone / DiscoursPermet la conversion voix-texte pour les messages d'alerte personnalisés
ContactsVous permet de sélectionner des destinataires dans le carnet d'adresses de votre téléphone
NotificationsAffiche une confirmation lorsqu'une alerte est envoyée
SMS (Android uniquement)Envoie des alertes directement depuis votre carte SIM
État du téléphone (Android uniquement)Détecte la disponibilité cellulaire pour une solution de secours intelligente

Vous pouvez accorder ou ignorer n'importe quelle autorisation lors de la configuration et les ajuster ultérieurement dans les paramètres système de votre appareil.

2. Aperçu de l'écran d'accueil

L'écran d'accueil comporte trois pages dans lesquelles vous pouvez naviguer en faisant glisser votre doigt vers la gauche ou la droite :

Page Nom Description
← Gauche Minuterie d'enregistrement Minuterie d'homme mort (Premium uniquement)
Centre Mode panique Gros boutons SOS et Je vais bien – la valeur par défaut
Droite → Mode standard Emplacements de messages d'alerte préconfigurés

L'application s'ouvre toujours en mode panique (page centrale).

3. Mode panique

La page centrale est conçue pour une simplicité maximale – idéale pour les urgences et les utilisateurs seniors.

  • Bouton SOS : envoie une alerte "SOS, j'ai besoin d'aide" avec votre position.
  • Bouton Je vais bien : envoie un statut rassurant « Je vais bien » à tous les contacts.
  • Boutons vocaux : appuyez sur l'icône du microphone à côté de SOS ou Je vais bien pour enregistrer un message vocal. Votre discours est transcrit en texte et inclus dans l'alerte.
  • Bouton d'appel d'urgence : compose immédiatement votre numéro d'urgence local (par exemple, 112, 911), en fonction des paramètres de votre pays.

4. Mode standard

Faites glisser votre doigt vers la droite depuis le mode Panique pour accéder au mode Standard. Cette page affiche une liste de créneaux d'alerte préconfigurés.

  • Emplacements de texte — jusqu'à 7 emplacements de message personnalisables (par défaut : 3). Appuyez sur n'importe quel emplacement pour modifier le message. Appuyez sur l'icône emoji pour changer l'icône du bouton.
  • Emplacement vocal – toujours disponible comme dernier emplacement. Appuyez dessus pour enregistrer un message vocal en temps réel ; le texte transcrit est envoyé comme alerte.

Le nombre d'emplacements de texte visibles peut être ajusté dans Paramètres > Nombre d'alertes configurées.

5. Gestion des destinataires

Appuyez sur l'icône des personnes dans la barre d'outils supérieure pour ouvrir l'écran Destinataires.

  • Ajouter manuellement : appuyez sur Ajouter et saisissez un numéro de téléphone au format international (par exemple, +31600000000).
  • Ajouter à partir des contacts : appuyez sur l'icône des contacts pour effectuer une sélection directement dans le carnet d'adresses de votre téléphone.
  • Ajouter un e-mail : ajoutez éventuellement une adresse e-mail à chaque destinataire pour la redondance des e-mails.
  • Supprimer : faites glisser ou appuyez sur l'icône de suppression à côté de n'importe quel contact.

Niveau gratuit : jusqu'à 1 destinataire. Premium : jusqu'à 50 destinataires.

Astuce iOS : si vous ne voyez que quelques contacts, accédez à Paramètres > Confidentialité > Contacts > TapAlert et sélectionnez Tous les contacts.

6. Envoi d'une alerte

  1. Ouvrez l'application (ou appuyez sur le widget de votre écran d'accueil).
  2. Appuyez sur le bouton d'alerte (SOS, je vais bien ou un emplacement en mode standard).
  3. Si un compte à rebours est configuré, une boîte de dialogue apparaît vous donnant le temps d'annuler.
  4. L'application récupère votre position GPS actuelle et crée le message.
  5. Le son d'alerte est émis pendant l'envoi.
  6. Un rapport d'état s'affiche avec les détails de réussite ou d'échec par destinataire.

Le message d'alerte comprend :

  • Votre nom (si défini dans Paramètres)
  • Le texte du message
  • Un lien Google Maps vers votre position actuelle
  • Votre niveau de batterie

7. Boîte de dialogue Compte à rebours

Avant l'envoi, TapAlert affiche un compte à rebours pour éviter les alertes accidentelles.

  • Durée par défaut : 10 secondes. Réglable de 0 (immédiat) à 30 secondes dans Paramètres > Durée du compte à rebours.
  • Annuler : appuyez sur ANNULER à tout moment pendant le compte à rebours pour abandonner.
  • Envoyer immédiatement : appuyez sur ENVOYER MAINTENANT pour ignorer le délai restant.

8. Minuterie d'enregistrement (Premium)

Le Check-in Timer est un interrupteur d'homme mort : si vous ne l'annulez pas à temps, TapAlert envoie automatiquement une alerte. Utile pour faire de la randonnée, voyager seul ou toute situation dans laquelle vous souhaitez que quelqu'un soit averti si vous cessez de répondre.

Pour commencer :

  1. Faites glisser votre doigt vers la gauche jusqu'à la page Minuterie d'enregistrement.
  2. Utilisez les sélecteurs de roues pour définir une durée (heures et minutes).
  3. Tapez éventuellement un message personnalisé.
  4. Appuyez sur Démarrer la minuterie.

En cours d'exécution :

  • Le compte à rebours s'affiche à l'écran.
  • Lorsqu'il reste moins de 5 minutes, TapAlert émet un son d'avertissement et affiche une boîte de dialogue :
    • Annuler : ignorez l'avertissement et laissez le minuteur continuer.
    • Arrêter : abandonnez le minuteur (aucune alerte ne sera envoyée).
    • Redémarrer - réinitialisez la minuterie à la durée d'origine.
  • Si le minuteur atteint zéro sans être arrêté ou réinitialisé, l'alerte est envoyée automatiquement.

Android : le minuteur fonctionne comme un service de premier plan et reste actif lorsque l'application est en arrière-plan ou que l'écran est éteint.

iOS : le minuteur peut ne pas se déclencher si l'application est fermée de force. Gardez l'application exécutée en arrière-plan.

9. Historique des alertes

Appuyez sur l'icône de l'horloge dans la barre d'outils supérieure pour afficher votre historique d'alertes.

  • Affiche l'horodatage, le message et la méthode de livraison pour chaque alerte.
  • Appuyez sur n’importe quelle entrée pour obtenir des détails sur la réussite/l’échec de chaque destinataire et des informations sur les erreurs.
  • Niveau gratuit : stocke les 20 dernières alertes. Premium : stocke jusqu'à 200.
  • Supprimez des entrées individuelles ou effacez tout l'historique de l'icône de la corbeille.

10. Paramètres

Appuyez sur l'icône d'engrenage pour ouvrir les paramètres.

Méthode SMS

Androïde:

  • SMS directs : envoie via votre carte SIM, pas besoin d'Internet. Préféré lorsque le cellulaire est disponible.
  • Twilio — envoie via l'API Twilio sur Internet (Wi-Fi ou données). Utilisé comme solution de secours automatique en cas d'échec de Direct SMS.

IOS :

  • Messages iOS (manuel) : ouvre l'application Messages native pré-remplie. Vous appuyez sur Envoyer manuellement. Aucun compte Twilio n'est nécessaire.
  • Twilio (Automatique) — envoie automatiquement via l'API Twilio. Nécessite un compte Twilio.

Pour les instructions de configuration de Twilio, consultez le Guide Twilio dans le menu Aide.

Redondance des e-mails

Configurez la messagerie SMTP pour envoyer des alertes par e-mail en plus des SMS. Utile lorsque les SMS ne sont pas disponibles mais que le Wi-Fi est connecté.

  1. Accédez à Paramètres > Redondance des e-mails.
  2. Activez la redondance des e-mails.
  3. Sélectionnez un fournisseur prédéfini (Gmail, Outlook, Yahoo, iCloud) ou saisissez des détails SMTP personnalisés.
  4. Entrez votre adresse e-mail et un mot de passe d'application (recommandé plutôt que votre mot de passe principal).
  5. Appuyez sur Tester la connexion par e-mail pour vérifier avant de vous y fier.

Les e-mails sont envoyés à tout destinataire de votre liste dont l'adresse e-mail est enregistrée.

Emplacement

  • Inclure l'emplacement : ajoute les coordonnées GPS (lien Google Maps) à chaque alerte.
  • Inclure l'adresse - tente également de résoudre une adresse postale à partir de vos coordonnées.

Méthode de déclenchement

  • Appuyez pour envoyer : une simple pression démarre le compte à rebours.
  • Appuyez et maintenez (5 s) — vous devez maintenir le bouton enfoncé pendant 5 secondes. Recommandé pour éviter les envois accidentels.

Compte à rebours des alertes

Définissez le délai avant l’envoi de l’alerte (0 à 30 secondes). Un compte à rebours plus long vous donne plus de temps pour annuler s'il est déclenché par erreur.

Paramètres sonores

  • Choisissez entre des sons simples ou des sons riches (ou aucun).
  • Modèle sonore : sélectionnez parmi Alerte rapide, bips courts, carillon musical, sirène lamentable, sonnerie à impulsions, orgue à tuyaux, anneau métallique ou ruche électronique.
  • Hauteur sonore : faible, standard ou élevée.
  • Volume — ajustez avec le curseur.
  • Les alertes SOS et Je vais bien ont des modèles sonores indépendants.

Autres paramètres

  • Votre nom : inclus dans les messages sortants afin que les destinataires sachent qui a envoyé l'alerte.
  • Durée du compte à rebours — délai avant l'envoi (voir ci-dessus).
  • Nombre d'alertes configurées : combien d'espaces de texte apparaissent en mode standard (3 à 7).
  • Démarrage automatique au lancement : déclenche le compte à rebours immédiatement à l'ouverture de l'application. Activez-le uniquement si vous lancez TapAlert via un raccourci de bouton physique.
  • Flash en cas d'alerte : fait clignoter l'écran et la torche lorsqu'une alerte est envoyée.
  • Pays d'origine : garantit que les numéros de téléphone locaux sont correctement formatés.
  • Langue de l'application : remplace la langue de l'interface (par défaut : langue de l'appareil).
  • Thème de l'application : Clair, Sombre ou Système par défaut.

11. Widget de l'écran d'accueil (Android)

Ajoutez un widget TapAlert à votre écran d'accueil Android pour un accès en un seul clic sans ouvrir l'application.

Pour ajouter :

  1. Accédez à Paramètres > Widget > Ajouter un widget.
  2. Choisissez Standard (1x1) pour une icône compacte ou Large (2x2) pour une meilleure visibilité.
  3. Suivez les instructions à l'écran pour le placer sur votre écran d'accueil.

Comment ça marche :

  • Appuyer sur le widget ouvre l'application et démarre immédiatement le compte à rebours de l'alerte.
  • Le son est joué et la boîte de dialogue du compte à rebours apparaît – vous pouvez toujours annuler en cas de déclenchement par erreur.

12. Faire ressortir les alertes sur les téléphones des destinataires

Vos destinataires peuvent configurer leur téléphone pour donner aux messages TapAlert une sonnerie spéciale ou contourner Ne pas déranger.

Destinataires iPhone :

  1. Ouvrez Contacts et recherchez votre entrée.
  2. Appuyez sur Modifier > Tonalité du texte.
  3. Activer le contournement d'urgence : cela contourne le mode silencieux et Ne pas déranger.
  4. Choisissez un ton fort et distinctif.

Destinataires Android :

  1. Ouvrez l'application Messages et recherchez la conversation.
  2. Appuyez sur le menu > Détails > Notifications.
  3. Définissez la priorité sur Élevée ou Urgente.
  4. Activez le remplacement de Ne pas déranger.

Vous utilisez Twilio ? Vos contacts recevront des messages de votre numéro Twilio ou de votre identifiant d'expéditeur (par exemple, TapAlert). Ils doivent ajouter ce numéro/ID à leurs contacts pour que les paramètres ci-dessus s'appliquent.

13. Configuration à distance (Magic Link)

Un membre de la famille ou un soignant peut configurer TapAlert sur votre appareil à distance à l'aide d'un Magic Link.

Le soignant ouvre TapAlert, sélectionne Configurer pour quelqu'un d'autre, remplit les contacts, les modèles de message et les informations d'identification Twilio/e-mail facultatives, puis génère un lien de configuration. Lorsque vous appuyez sur ce lien sur votre appareil, TapAlert applique automatiquement la configuration complète.

Cela signifie qu'un membre de la famille féru de technologie peut tout configurer sans avoir besoin d'être présent : il vous suffit d'appuyer sur un lien.
