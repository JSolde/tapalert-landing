1. Problèmes d'envoi de SMS

1.1 Direct SMS Pas d'envoi (Androïde)

Symptôme : Vous appuyez sur "Envoyer une alerte", mais le SMS échoue. L'historique des alertes affiche une erreur pour un ou plusieurs destinataires.

  • Aucun signal cellulaire : vérifiez la barre d'état. La page Statut de l'application affiche votre statut cellulaire actuel.
  • Pas de carte SIM : Direct SMS nécessite une carte SIM fonctionnelle. Vérifiez l'état de la carte SIM sur la page État.
  • Autorisations : Assurer L'autorisation SMS est autorisée dans les paramètres du téléphone.
  • Format non valide : utilisez le format international avec l'indicatif du pays (par exemple, +34600123456).

1.2 SMS Twilio non envoyé

Symptôme : l'alerte échoue avec un message d'erreur Twilio.

  • Internet requis : Twilio a besoin d'une connexion Wi-Fi ou de données cellulaires.
  • Identifiants : vérifiez le SID du compte, le jeton d'authentification et le numéro de téléphone Twilio dans les paramètres.
  • Limites de l'essai : les compte d'essai ne peuvent envoyer qu'aux numéros de téléphone vérifié.
  • Solde : vérifiez votre solde Twilio compte.

1.3 Problèmes avec le compositeur de SMS iOS

Symptôme : sur iPhone, l'éditeur de SMS s'ouvre mais le message n'est pas transmis.

  • Signal : l'application Messages a besoin d'un signal cellulaire (ou d'appels WiFi si elle est active).
  • Annulation : si vous appuyez sur Annuler dans le composeur, l'alerte est annulée.
  • Numéro invalide : iOS mettra le message en file d'attente et échouera silencieusement. Vérifiez l'application Messages pour les points d'exclamation rouges.

2. Problèmes de redondance des e-mails

2.1 Échec du test de courrier électronique

  • Paramètres SMTP : vérifiez l'hôte et le port (Gmail/iCloud utilise 587).
  • Mot de passe de l'application : la plupart des fournisseurs exigent un Mot de passe de l'application si 2FA est activé.
  • Internet : la messagerie électronique nécessite une connexion de données active.

3. Problèmes Emplacement

3.1 Emplacement manquant dans l'alerte

  • Autorisations : Assurer Emplacement l'autorisation est définie sur "Pendant l'utilisation" ou "Toujours".
  • GPS désactivé : vérifiez que les services de localisation du téléphone sont activés.
  • Réception : à l'intérieur ou dans des zones denses, un délai d'attente du GPS peut se produire.

4. Connectivité et statut

4.2 Signification des icônes de bouclier

  • Vert : Prêt pour une livraison fiable de SMS/Twilio.
  • Orange : Incertain. WiFi uniquement, pas de Twilio. Les appels WiFi peuvent fonctionner.
  • Rouge : Déconnecté. Le cellulaire et le WiFi sont désactivés.

5. Problèmes de minuterie d'enregistrement

5.1 La minuterie ne se déclenche pas

  • Application fermée : évitez de forcer la fermeture de l'application. Sur iOS, vous doit appuyez sur la notification pour la déclencher.
  • Optimisation de la batterie : définissez TapAlert sur "Unrestricted" dans les paramètres de la batterie Androïde.
  • Notifications : Assurer les autorisations de notification sont accordées.

6. Problèmes de voix-texte

6.1 La reconnaissance vocale ne fonctionne pas

  • Autorisations : Assurer Les autorisations Microphone et Reconnaissance vocale (iOS) sont accordées.
  • Internet : la plupart des moteurs de reconnaissance nécessitent une connexion active pour le traitement.
  • Incorrect Langue : La reconnaissance utilise le paramètre de langue actuel de l'application. Vérifiez **Paramètres > Langue**.

7. Contacts et problèmes de destinataires

7.1 Impossible de voir Contacts

  • Accès limité à iOS : si vous avez précédemment choisi "Sélectionner Contacts...", accédez à **Paramètres > Confidentialité et sécurité > Contacts > TapAlert** et remplacez par **"Tous Contacts"**.
  • Autorisations Androïde : Assurer **Contacts** L'autorisation est définie sur "Autoriser" dans les paramètres système.
  • Format : les nombres Assurer incluent l'indicatif du pays avec un préfixe **+**.

8. Problèmes d'achat via l'application

8.2 Acheté mais affiche toujours Gratuit

  • Restaurer les achats : dans l'écran de mise à niveau, appuyez sur **Restaurer**bouton pour revalider votre achat auprès du magasin.
  • Même compte : Assurer vous êtes connecté au même magasin compte que celui utilisé pour l'achat initial.

9. Problèmes de widgets (Androïde)

9.1 Le widget n'apparaît pas

  • Ajout manuel : essayez d'ajouter le widget depuis **Paramètres > Widget > Ajouter** dans l'application s'il n'apparaît pas dans le sélecteur de widgets système.
  • Optimisation de la batterie : une optimisation agressive peut empêcher le widget de lancer correctement l'application.

10. Problèmes de notifications

10.2 Les destinataires ne remarquent pas les appels/SMS

  • iPhone : demandez aux destinataires d'activer **Contournement d'urgence** pour votre numéro de contact.
  • Androïde : demandez aux destinataires de définir la priorité de notification sur **"Haut"** ou **"Urgent"** pour vos messages.

11. Problèmes de comportement des applications

11.1 Une alerte se déclenche immédiatement en cas d'ouverture

Accédez à **Paramètres** et désactivez **"Démarrage automatique au lancement"**. Cette fonctionnalité est destinée aux paniquebouton physiques.

12. Problèmes spécifiques à iOS

12.1 Twilio contre Messages iOS

  • Twilio (Automatique) : Envoi silencieux en arrière-plan, nécessite Internet.
  • Messages iOS (Manuel) : vous oblige à appuyer sur Envoyer dans le compositeur, fonctionne via le cellulaire.

13. Androïde-Problèmes spécifiques

13.1 Optimisation de la batterie

Certains fabricants (Xiaomi, Samsung, Huawei) exigent des étapes manuelles pour éviter que l'application ne soit supprimée en arrière-plan. Vérifiez les paramètres de la batterie de votre téléphone pour **"Aucune restriction"**.

14. Conseils généraux

✅ Avant une urgence

  • Envoyez une alerte de test pour vérifier la configuration.
  • Vérifiez la page État pour la connectivité.
  • Assurer toutes les autorisations sont accordées.
